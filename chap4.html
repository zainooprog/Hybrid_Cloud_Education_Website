Chapter 4: Cloud Architecture & Security Layers

IaaS

Renting virtual servers and networking. You manage the OS. Example: Amazon EC2

PaaS

Focus on code without managing servers. Example: Google App Engine

SaaS

Ready-to-use software accessed via browser. Example: Gmail, Microsoft 365

Cloud Security: Shared Responsibility

Security OF the Cloud: Cloud provider handles hardware, cooling, physical security.

Security IN the Cloud: User is responsible for passwords, encryption, access management.
